## Large-Color-Wheel
Easy to see huge Color code wheel

Written in Kotlin, Need Java8+ to run

This project base on https://github.com/ohmae/wheel-color-chooser

## How to compile
install kotlin and run
```
kotlinc src   -include-runtime -d  app.jar
```

## How to run
```
javaw -jar app.jar
```
